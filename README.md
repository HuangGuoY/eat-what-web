# 吃货的后台
用vue admin template模板做的一个后台

**在线 demo:** http://panjiachen.github.io/vue-admin-template

#运行方式
# 克隆项目
git clone https://github.com/HuangGuoY/eat-what-web.git

# 进入项目根目录
cd eat-what-web

# 运行npm install
npm install

# 启动
npm run dev
